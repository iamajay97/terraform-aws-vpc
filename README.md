# Terraform AWS VPC Infrastructure

## Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform to provision AWS networking resources.

## Resources Created

* VPC
* Public Subnet
* Internet Gateway
* Route Table
* Route Table Association

## Technologies Used

* Terraform
* AWS
* Infrastructure as Code (IaC)

## Project Structure

terraform-aws-vpc/
├── provider.tf
├── variables.tf
├── main.tf
├── outputs.tf
└── README.md

## Purpose

This project was created to demonstrate Terraform fundamentals and AWS networking concepts used in DevOps and Cloud environments.
